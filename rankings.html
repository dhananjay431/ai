05

Performance

Performance rankings compare how fast and reliably models respond. A model with a slightly lower benchmark score may still be the best production choice if it is cheaper, faster, and available at high throughput.

  • Latency: time to first token and total completion time.
  • Throughput: output tokens per second under normal load.
  • Reliability: stable responses, low error rate, and predictable formatting.
  • Price-performance: quality per dollar for chat, coding, RAG, and batch workflows.

09

Context Length

Context length rankings compare how much text, code, or multimodal input a model can process in one request. Longer context is useful for legal documents, research papers, codebases, logs, transcripts, and enterprise knowledge bases.

  • Short context: fast, cheap, ideal for chat and small tasks.
  • Medium context: good for RAG snippets, product docs, and conversations.
  • Long context: handles files, transcripts, and multi-document analysis.
  • Very long context: useful for repository-scale and archive-scale workflows, but needs careful evaluation.
10

Tool Calls

Tool-call rankings measure how well models decide when to call a function, choose valid arguments, chain multiple tools, recover from errors, and return a final user-friendly answer. This matters for agents, automation, data extraction, booking flows, and API-driven assistants.

1
Schema Accuracy

Outputs valid JSON and respects required fields.

2
Tool Selection

Chooses the correct API or refuses unnecessary calls.

3
Recovery

Handles missing data, failed calls, retries, and follow-up prompts.

Audio Input

Audio input rankings represent models and apps that process spoken prompts, meetings, interviews, lectures, voice commands, and conversational audio. The best systems combine transcription, understanding, summarization, and action-taking.

  • Voice assistants: real-time user interaction and command handling.
  • Meeting intelligence: summaries, action items, and speaker-aware notes.
  • Education: lecture understanding, tutoring, and pronunciation feedback.
  • Support: call analysis, routing, and agent-assist workflows.
